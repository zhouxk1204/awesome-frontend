# React 简介

## 📝 概要

React 是一个用于构建用户界面的 JavaScript 库，以组件化、虚拟 DOM、单向数据流为核心。

## 📌 核心知识点

- 组件化：每个组件拥有自己的逻辑与状态。

- 虚拟 DOM：最小化真实 DOM 更新，提高性能。

- 单向数据流：数据始终从父组件流向子组件。

- JSX：让开发者可以在 JS 中写类似 HTML 的结构。

## 💻 示例代码

```jsx
function App() {
  return <h1>Hello, React</h1>;
}
```

## ⚠️ 常见误区

- React 不是 MVC 框架，仅负责 UI 层。

- 虚拟 DOM ≠ 一定更快，优势主要在于“可预测且可控”。

## 🎯 适用场景

- 构建复杂前端应用（SPA）

- 有长期维护需求的项目
