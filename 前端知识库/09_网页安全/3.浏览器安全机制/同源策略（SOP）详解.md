# 同源策略（SOP）详解

## 📝 概要

同源策略是浏览器最核心的安全边界，决定“能不能读”，而不是“能不能发”。

## 📌 核心知识点

- 协议 + 域名 + 端口 必须一致
- JS 跨域受限，但请求可发
- CORS 是对 SOP 的授权例外

## 💻 示例代码

```text
https://a.com  ❌  http://a.com
https://a.com  ❌  https://api.a.com
```

## ⚠️ 常见误区

- 以为跨域请求不能发送

- 混淆 SOP 与 CORS

- 认为 Cookie 读取受 SOP 控制

## 🎯 适用场景

- 浏览器安全理解

- XSS / CSRF 根因分析

- 跨域方案设计
