# Object.is 同值相等

## 📝 概要
Object.is() 用于比较两个值是否“同值相等”，解决 === 的两个特殊问题：NaN 和 ±0。

## 📌 核心知识点
- NaN 与 NaN 被认为相等
- 可以区分 +0 和 -0
- 其他情况下与 === 相同

## 💻 示例代码
```js
Object.is(NaN, NaN); // true
Object.is(-0, +0);   // false
Object.is(0, 0);     // true
```

## ⚠️ 常见误区
- 误以为 Object.is() 会替代 ===
- 忽略仅在 NaN 和 ±0 上有区别

## 🎯 适用场景
- 检查 NaN
- 区分 ±0 的高精度计算
- 需要严格比较特殊值的场景
