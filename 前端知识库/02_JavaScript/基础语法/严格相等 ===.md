# 严格相等 ===

## 📝 概要
严格相等运算符 === 在比较两个值时，不仅要求值相等，还要求类型相同。

## 📌 核心知识点
- 值和类型必须相同，才会返回 true
- 不进行类型转换
- NaN !== NaN，-0 === +0

## 💻 示例代码
```js
'1' === 1;        // false
true === 1;       // false
null === undefined; // false
0 === -0;         // true
```

## ⚠️ 常见误区
- 误以为 === 可以区分 NaN
- 误以为 === 可以区分 +0 和 -0

## 🎯 适用场景
- 大多数日常值比较
- 保证类型安全的判断
